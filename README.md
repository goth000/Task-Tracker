# Трекер задач (Todo Tracker) — Итоговая лабораторная работа

## Описание
Консольное приложение для управления задачами (вариант 7). Хранит данные в JSON-файле `data/data.json`. Поддерживает CRUD операции, фильтрацию по группам, отчёт о просроченных задачах.

## Структура данных (JSON)
[
{
"id": "1",
"title": "first",
"due": "2025-12-29",
"priority": "low",
"group": "",
"done": false
}
]



## Сборка и запуск

### Windows (MinGW)
g++ src/lab5_Danil.cpp -o todo.exe -std=c++17 -O2
./todo.exe



### Visual Studio
1. Открой `src/lab5_Danil.cpp`
2. Build → Build Solution (Ctrl+Shift+B)

## Демо сценарий
Создать задачу (приоритет: low, название, дата, группа)
​

Просмотр всех задач

Отчёт просроченных (ввести дату 2025-12-24)

Фильтр по группе

Удалить задачу №1
​



## Меню программы
1 - создать задачу
2 - удалить задачу
3 - изменить задачу
4 - фильтр по группе
5 - отчёт о просроченных задачах
6 - просмотр всех задач
